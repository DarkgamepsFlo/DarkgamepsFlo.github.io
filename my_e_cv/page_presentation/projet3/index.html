Le Jeu Quarto en JAVA

Ce projet consiste à réaliser un jeu de société dans le langage qu'on voulait, j'ai choisi le jeu quarto et le langage JAVA pour le réaliser. Au tout départ, j'ai réalisé l'ensemble du jeu dans le terminal de commande pour pouvoir avoir une base assez solide pour pouvoir par la suite réaliser l'interface graphique.

Le quarto est un jeu ressemblant un peu au morpion mais avec un peu plus de condition. Nous devons réaliser une ligne de quatre pièces de même caractéristique. Il existe plusieurs types de caractéristiques : carré, rond, grand, petit, avec un trou, sans trou, blanc et marron.

Une fois le jeu dans le terminal effectué, j'ai utilisé la Librairie MG2D en JAVa pour pouvoir réaliser l'interface graphique en utilisant les mêmes fonctions que dans la version effectuée dans le terminal.

Par la suite plusieurs fonctionnalités était imposé. Nous avons en premier la mise en place d'une IA. Cette dernière doit posséder plusieurs niveaux et donc plusieurs manières d'analyser le jeu en cours.

Ensuite il fallait mettre en place la mise en ligne du jeu. J'ai réussi à faire en sorte que ma machine puisse simuler deux personnes venant d'un port différent et ces deux personnes sont gérés par un serveur qui va permettre d'effectuer l'ensemble des échanges pendant le jeu.

Enfin une base de données doit être présente et doit permettre de récupérer des informations sur l'ensemble des parties et ici, celle-ci est effectué en MySQL. Pour pouvoir respecter les rèles, je prends dans le jeu uniquement les pseudos des joueurs ainsi que le nombre de fois où ilss gagnent, rien de plus. Ces données peuvent permettre de faire un classement des meilleurs joueurs en affichant leurs pseudos.
